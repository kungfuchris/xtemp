Walkhub
=======

Prerequisites

Git
Drush

Installation

  Go to a temporary folder
  $ cd /tmp

  Copy and paste this into your shell to install everything:
  $ curl -L -s http://goo.gl/kHtmHH | bash

  Go inside the distro folder
  $ cd walkhub
  
  Run install
  $ ./install.sh /path/to/drupal/root
